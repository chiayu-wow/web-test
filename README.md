### php mailer 套件

#### 透過php傳送郵件到註冊用戶

須先載入 composer in your computer 
<https://getcomposer.org/download/>


接著藉由composer 來下載php mailer套件


<https://github.com/PHPMailer/PHPMailer>

```
composer require phpmailer/phpmailer
```
